## Run Locally

Clone the project

```bash
  git clone https://github.com/SrijanDas/social-media-app.git
```

Go to the project directory

```bash
  cd social-media-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
